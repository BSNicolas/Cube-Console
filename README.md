# Cube-Console

Console de jeu que j'ai réalisé durant mon semestre à l'international à Amsterdam.   
Dans ce repository il y a surtout de la documentation en anglais par rapport au projet :   
L'architecture du projet est visible dans le fichier `system_architecture.md`.   
Le schéma de la console et les éléments qui la constitue se trouvent dans le fichier `technical_documentation.md`.   
Dans les fichiers `uxd/concept_design.md`, `uxd/conclude_reflect.md`, `uxd/create_test.md` se trouvent des visuels et informations sur la console.   

Il y a du code avec le fichier `cube_invaders.ino`. Dans l'état il ne fonctionne pas, il faut remplacer certaines variables. Le code est ici présent à titre d'exemple.   

# Principe de la console

- Il s'agit d'une console dont le boitier à été réalisé avec une découpeuse laser et le jeu avec du code c++ pour Arduino
- Le jeu jouable dessus est une sorte de space invaders
